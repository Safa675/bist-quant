# Settings

Shared path helpers, output directory resolution, and environment variable parsing
primitives used by the library.

## Key modules

- `bist_quant.runtime` — data/regime path resolution (`BIST_DATA_DIR`, etc.)
- `bist_quant.common.data_paths` — canonical data file locations
- `bist_quant.common.cache_config` — disk cache TTL configuration
